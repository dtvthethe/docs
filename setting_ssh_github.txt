1/ Generate SSH key:
	ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

2/ Press `Enter`

3/ Input Type passphrase

4/ Copy SSH key:
	cat ~/.ssh/id_rsa.pub
or with Windows:
	clip < ~/.ssh/id_rsa.pub
	then
	ssh-add ~/.ssh/id_rsa

5/ Config Multiple SSH:
$ nano ~/.ssh/config
Host github.com
  HostName github.com
  IdentityFile ~/.ssh/id_rsa
Host <IP 1>
  HostName <IP 1>
  IdentityFile ~/.ssh/id_rsa_ip1
Host <IP 2>
  HostName <IP 2>
  IdentityFile ~/.ssh/id_rsa_ip2
